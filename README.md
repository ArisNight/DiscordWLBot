# Discord WhiteList Bot

This bot allows you to accept and process applications for private minecraft servers. 

The DiscordSRV plugin and the enabled function of broadcasting the server console to the Discord channel are required for correct operation. 

THE VALUE OF THE <DiscordChatChannelBlockBots> FUNCTION MUST BE SET TO TRUE IN THE DiscordSRV CONFIGURATION.
